A plugin to generate and display various types of fractals using WebGL.

== Description ==

The Fractal Generator plugin allows users to generate and explore different types of fractals directly on your WordPress site. It utilizes WebGL to render complex fractal images in real-time, offering a smooth and interactive experience.

**Features:**
- Supports various types of fractals including Mandelbrot, Julia, Burning Ship, and Magnet Fractals.
- Allows customization of fractal parameters and color schemes.
- Provides real-time zoom and pan capabilities.
- Includes a user-friendly interface with sliders and buttons for easy control.

== Installation ==

1. Upload the `fractal-generator` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Add the shortcode `[fractal_generator]` to any post or page where you want the fractal generator to appear.

== Frequently Asked Questions ==

= How do I change the type of fractal displayed? =
Use the dropdown menu labeled "Liste de fractales" to select the desired fractal type.

= Can I customize the colors of the fractal? =
Yes, you can choose from predefined color schemes or create a custom color scheme using the provided color pickers.

= How do I zoom in and out of the fractal? =
You can use the mouse wheel to zoom in and out. On mobile devices, you can use pinch gestures to zoom.
